# Canonical ID Specification

All scripture references use the following format:

book-chapter-verse

Example:

genesis-001-001
john-003-016
revelation-022-021

Rules:

• book names are lowercase
• hyphen separated
• chapter and verse are zero padded
• canonical source: canon/books/index.json

Entities use:

entity-{name}

Examples:

entity-adam
entity-moses
entity-seraphim

Themes use:

theme-{name}

Examples:

theme-covenant
theme-kingdom
theme-salvation
