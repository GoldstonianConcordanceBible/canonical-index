# GCB Architecture
## v3.0 Layers 

The Goldstonian Concordance Bible dataset is composed of five layers:

1 Canonical Registry
2 Verse-Level Dataset
3 Knowledge Graph
4 Concordance Engine
5 AI Training Corpora

## v2.0 Layers
1. Canon Layer
2. Ontology Layer
3. Entity Layer
4. Passage Layer
5. Media Layer
6. Search Layer
7. API Layer
8. Evaluation Layer
9. Export Layer

## Core design principle
Every answerable biblical question should be routable through:
question → concept → source bundle → media bundle → benchmarkable output
