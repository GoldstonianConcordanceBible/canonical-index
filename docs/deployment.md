# Deployment

## Current deployment path
GitHub Pages serves the public site and static export artifacts.

## Deployment flow
1. Push to main
2. Run validation scripts
3. Build public artifacts
4. Publish _site to GitHub Pages

## Public outputs
- homepage
- search index
- llms.txt
- sitemap.xml
- export bundles
