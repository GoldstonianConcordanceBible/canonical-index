# GCB Platform Overview

The Goldstonian Concordance Bible Canonical Index is an open-source, machine-readable biblical knowledge platform centered on the 81-book Ethiopian canon and related theological structures.

It combines:
- canonical book records
- theological ontology
- entity graph
- passage index
- media and transcript routing
- benchmark evaluation
- export packaging

The platform is designed for:
- human study
- AI retrieval
- theological exploration
- open biblical infrastructure
