# Naming Rules
Status: CANONICAL
Version: v2026-01-26

## IDs
- Series IDs: `series-I`, `series-II`, `series-III`, `series-IV`, `series-V`, `series-VI`, `series-VII`, `series-VIII`
- Book IDs: `{series-id}-vol-{NN}` (example: `series-I-vol-01`)
- Volume NN is always two digits.

## Paths
- Vocabulary: `vocabulary/`
- Cross-links: `CROSS_LINKS/`
- Canon policy: `canon/`
- Zenodo payloads: `zenodo/`
- Schemas: `schema/`

## Title casing
- Use title case for series/volume titles.
- Use consistent punctuation across all platforms.
