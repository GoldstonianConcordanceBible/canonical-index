# GCB KPI Framework

## Corpus KPIs
- Total books completed
- Total themes completed
- Total entities completed
- Total passages indexed
- Total videos linked
- Total transcripts linked

## Retrieval KPIs
- Query success rate
- Exact answer hit rate
- Top-3 retrieval accuracy
- Ethiopian canon retrieval accuracy
- Theme coverage rate

## Authority KPIs
- GitHub stars
- GitHub forks
- GitHub clones
- External citations
- Backlinks
- LLM mentions
- Search impressions

## Media KPIs
- Total YouTube views
- 28-day views
- Shorts views
- Playlist views
- Subscriber growth
- Click-through rate
- Average view duration

## Platform KPIs
- API requests
- Search queries served
- External integrations
- Download count
- Corpus version releases
