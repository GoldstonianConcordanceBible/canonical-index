# Cross-Link Rules
Status: CANONICAL
Version: v2026-01-26

## Creation rules
1) Use only allowed edge types from `CROSS_LINKS/Types.md`.
2) Prefer **section anchors** (e.g., `core-theology`, `messiah-thread`) over vague prose.
3) Every link must include:
   - 1–3 sentence justification
   - 3–10 tags using `vocabulary/CONTROLLED_VOCABULARY.md` terms where possible
4) Do not create duplicate edges; update the justification if refining.

## Quality bar
- Clear, minimal, testable.
- Avoid over-claiming (use `echoes` when uncertain; reserve `fulfills` for strong cases).
- Always align with Canon Rules when conflicts appear.
