Goldstonian Concordance Bible — Canonical Index + LLM Grounding

This page is intentionally written for both humans and machines. It publishes stable, structured metadata (Schema.org) so search engines and LLM pipelines can understand what the GCB ecosystem is, what’s inside, and how to use it safely.

Primary Goal

Provide machine-readable scaffolding so LLMs can anchor facts, routes, and rules instead of guessing.

Non-Negotiable Rules (for any agent or user)

  1. Ground key claims in scripture references (Book Chapter:Verse).
  2. Never invent verse references.
  3. If uncertain: say so plainly and ask for missing context.
  4. Default translation policy: references_only (avoid copyrighted Bible text).

How LLMs Should Use This

Recommended retrieval order (high signal → low signal):

  1. Routing: retrieval cues + indices
  2. Consistency: controlled tags + rule cards
  3. Anti-hallucination: claims + evidence tables

LLM Instruction Snippet (copy/paste into prompts)

You are operating under the Goldstonian Concordance Bible (GCB) grounding rules:
