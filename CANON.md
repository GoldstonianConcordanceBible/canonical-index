# Canon

The Goldstonian Concordance Bible (GCB) Canonical Index primarily follows the **81-book Ethiopian Biblical canon**, which represents the most expansive traditional canon in Christian history.

This repository organizes biblical data using a machine-readable structure that allows comparison between multiple canonical traditions while preserving the Ethiopian canon as the primary reference framework.

---

## Canonical Traditions Represented

### Ethiopian Canon (Primary)

Total Books: **81**

This canon includes:

- Old Testament books
- New Testament books
- Deuterocanonical texts
- Ethiopian-distinct works such as:

Examples:

- 1 Meqabyan
- 2 Meqabyan
- 3 Meqabyan
- Jubilees
- Enoch

The Ethiopian canon provides the **primary canonical authority for GCB indexing.**

---

### Catholic Canon

Total Books: **73**

Includes:

- Deuterocanonical books such as:
  - Tobit
  - Judith
  - Wisdom
  - Sirach
  - Baruch
  - 1–2 Maccabees

---

### Protestant Canon

Total Books: **66**

Used widely in modern Protestant traditions.
